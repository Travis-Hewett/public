GeoLocation Estimation and Metadata Analysis

Find anyone's Location on Social Media Profile

what we do:

CantHide, finds previous locations by looking at a given social media account and other Social Media accounts.

About:

  • Using the EXIF data on the images from the social media accounts you have provided, CantHide carries out location detection. If the EXIF data is removed (as in the majority of social media applications), complicated DL models can estimate the photograph's location with great accuracy.
  • Using its own methods, CantHide finds the accounts of the users whose data you want to extract on various platforms and analyzes the accounts' truth.
  • By doing manipulation analysis on the photographs on the target user's profile, CantHide gives you a thorough output. The system's whitepaper and source code will shortly be made available to the public
